# SimpleWarikanApp

`SimpleWarikanApp`は、飲み会や旅行などで発生した金額を簡単に割り勘計算できるAndroidアプリです。シンプルなUIで直感的に操作でき、素早く会計を済ませることができます。

## 主な機能

- **合計金額の入力**: 支払った合計金額を入力します。
- **人数の入力**: 割り勘する人数を入力します。
- **割り勘結果の表示**: 一人当たりの金額と、割り切れない場合の端数を計算して表示します。

## 使用技術

- **UIフレームワーク**: Jetpack Compose
- **プログラミング言語**: Kotlin
